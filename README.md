# SkillForge AI 🚀

An AI-powered career development platform that helps users discover career paths, track learning progress, practice interviews, and receive personalized AI guidance.

SkillForge AI combines modern full-stack technologies with Gemini AI to provide an interactive learning and career preparation experience.

---

## 🌟 Features

### 🔐 Authentication & Security

* User registration and login
* JWT-based authentication
* Google OAuth login
* Protected routes
* Role-based access control (RBAC)
* Admin and User permissions

Roles:

* User
* Admin

---

# 👨‍💻 User Features

## Career Roadmaps

* Explore career learning paths
* View roadmap details
* Track learning progress
* Enroll in career paths
* Follow structured skill development plans

---

## AI Chat Assistant

An intelligent career assistant powered by Gemini AI.

Capabilities:

* Answer career-related questions
* Provide learning guidance
* Understand user context
* Give personalized suggestions

## AI Mock Interview

Practice interviews with AI.

Features:

* Generate interview questions
* Answer interview questions
* Receive AI evaluation
* Get strengths and weaknesses

* Interview performance scoring

## Interview Reports

* Communication score
